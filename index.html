Wp-magazeen-modern

Modernized wordpress magazeen theme to support new wordpress features

View project onGitHub

In February 2009, Smashing Magazine featured a beautiful theme designed by WeFunction Design Agency. The original theme can be found here http://www.smashingmagazine.com/2009/02/23/magazeen-free-magazine-look-wordpress-theme/. Unfortunately, the original theme hasn't been upgraded since then. It misses out on some of the cooler, new wordpress features. While searching for a new theme for Nokia Reviewz, we came across magazeen and decided this was the theme we'll be using. And hence, it has been upgraded to provide more features which have been available since the original theme released.

Updated Magazeen Theme for wordpress to support following additional features

  • Wordpress Menu which can be added from Dashboard → Appearance → Menu.
  • Header Image, which can be uploaded from Dashboard → Appearance → Header.
  • Image box instead of search button in the header, which can be used for image or ads.
  • Upload Image button added for article image on the New Post page. Now you don't have to manually copy the image url.
  • Support for using first image as a thumbnail image. This is useful for those who have lots of existing posts or don't want to manually set featured image manually everytime.
